# Raven
## 中文
Raven类似于一个实现了NAT穿透的简化版SSH，由C++编写。

- 使用AES加密。
- 支持文件传输。
- 使用P2P模式，有服务器中转模式作为替代
- 服务器为linux，客户端和主机均支持linux和macos。
